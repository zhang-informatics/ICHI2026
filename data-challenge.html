Overview

The Data Challenge Program at IEEE ICHI 2026 focuses on advancing human-AI collaboration in healthcare data science. This program showcases results from the AgentDS Healthcare benchmark, where participants developed reproducible solutions for realistic healthcare prediction tasks using collaborative human-AI workflows.

The session brings together researchers and practitioners to:

  • Present accepted papers from the Data Challenge track
  • Share insights into human-AI collaboration in real-world healthcare settings
  • Discuss design principles for reliable, reproducible, and high-impact healthcare data science systems

About AgentDS Healthcare

The program is based on the AgentDS Healthcare benchmark, which provides:

  • Realistic, domain-specific healthcare datasets
  • Multimodal and structured prediction tasks
  • Standardized evaluation protocols
Dataset AgentDS Healthcare dataset on Hugging Face. Domain Page Healthcare task details and AgentDS domain information. Technical Report AgentDS technical report on arXiv.
